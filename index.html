Features

Everything starts with Events and Actions.

Actions + Events

Create Actions, then trigger them from TikTok or Twitch events such as gifts, chat commands, follows, likes, shares, bits, and more.

Custom Overlays

Use goals, gift overlays, emoji effects, hype moments, wheel overlays, Magic 8 Ball, and other visual tools in OBS or TikTok Live Studio.

Local TTS

Use local/offline TTS workflows with Piper and viewer voice assignment options for more personalized stream interactions.

Song Requests

Manage a song queue, requests, commands, and playback controls from inside Hollow Bridge.

Streamer.bot + OBS

Connect to Streamer.bot and OBS WebSocket to trigger external actions, scene changes, and other stream automation.

Creator-Focused Tools

Designed around stream workflows instead of generic automation, with future support planned for more interactive creator tools.

OBS + TikTok Live Studio

Capture overlays your way.

OBS can use Browser Source URLs. TikTok Live Studio can use dedicated Hollow Bridge capture windows when browser links are unreliable.
