# Hollow Bridge Site

Official website and landing page for Hollow Bridge, a creator-focused live stream automation app for Twitch and TikTok.

## Cloudflare Workers deploy settings

Project name: `hollowbridge-site`

Deploy command:

```bash
npx wrangler deploy
```

Path:

```text
/
```

This repo uses `wrangler.toml` with static assets served from the repository root.
